# CPlayerDragging
Unturned Dragging Plugin

Features:

- Drag players around by looking at them and surrendering
- Dragged player enters the vehicle with you and cannot exit or swap seats ( you will not be able to enter the vehicle if there is not sufficent space for the both of you )
- Only allow dragging if the player has the dragging permission (configurable)
- Only allow dragging if the target is restrained
